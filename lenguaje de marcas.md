```## Lenguaje de Marcas (XML y HTML) ```

En esta asignatura, nos centramos en dos lenguajes fundamentales para el desarrollo web y la organización de información: **HTML** y **XML**. 
Comenzamos con **HTML (HyperText Markup Language)**, el lenguaje estándar para crear la estructura y el contenido de las páginas web. Aprendemos a usar etiquetas HTML para definir elementos como **encabezados**, **párrafos**, **listas**, **enlaces**, **formularios**, **tablas**, y más, organizando la información de manera clara y accesible para el usuario.
A continuación, exploramos **XML (eXtensible Markup Language)**, un lenguaje de marcado diseñado para almacenar y transportar datos. A diferencia de HTML, XML no define una presentación visual, sino que se enfoca en organizar los datos de forma jerárquica para que puedan ser fácilmente procesados por diferentes sistemas. Estudiamos cómo diseñar **documentos XML**, crear **esquemas (XSD)** para validar la estructura de los datos y explorar su aplicación en la interoperabilidad de sistemas. Al finalizar, los estudiantes tienen una comprensión completa de cómo usar estos lenguajes en proyectos web y en la integración de sistemas.
