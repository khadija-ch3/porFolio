```## Bases de Datos ```  

En esta asignatura, estudiamos el **almacenamiento y gestión de datos** en sistemas informáticos, comenzando con la organización de la información en **bases de datos**. Analizamos el proceso de **almacenamiento de ficheros** y cómo se utilizan diferentes estructuras de datos para organizar grandes cantidades de información de manera eficiente y segura. A continuación, aprendemos sobre los **Sistemas de Gestión de Bases de Datos (SGBD)**, herramientas que permiten crear, modificar y consultar bases de datos. Estudiamos sistemas populares como **MySQL**, **PostgreSQL** y **SQL Server**, y aprendemos cómo trabajar con **lenguaje SQL** (Structured Query Language) para realizar operaciones como **SELECT**, **INSERT**, **UPDATE**, y **DELETE** en bases de datos relacionales. Un aspecto clave de la asignatura es el **modelo entidad-relación (ER)**, que se utiliza para representar gráficamente las relaciones entre entidades en una base de datos. Aprendemos a crear diagramas ER y a convertir estos modelos en bases de datos relacionales, optimizando la estructura para asegurar una gestión eficiente de los datos.
